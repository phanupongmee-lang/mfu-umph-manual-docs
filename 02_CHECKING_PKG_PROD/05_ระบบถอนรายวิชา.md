# 📚 API Reference: Course Withdraw System

เอกสารรวบรวม Stored Procedure และ Function สำหรับระบบถอนรายวิชา (Course Withdraw Online System) แบ่งออกเป็น 3 Package หลัก ดังนี้

---

## 👤 1. นักศึกษา (`Withdraw_Student_pkg`)

### 📌 FUNC_GET_ACADYEAR

- **คำอธิบาย:** ปีปัจจุบันปฎิทินการลงทะเบียน
    
- **Parameters:**
    
    SQL
    
    ```
    VSTUDENTCODE IN NUMBER
    ```
    
- **Example:** `2568`
    

### 📌 FUNC_GET_SEMESTER

- **คำอธิบาย:** ภาคปัจจุบันปฎิทินการลงทะเบียน
    
- **Parameters:**
    
    SQL
    
    ```
    VSTUDENTCODE IN NUMBER
    ```
    
- **Example:** `1`
    

### 📌 FUNC_GET_STUDENTID

- **คำอธิบาย:** ตรวจสอบรหัส studentcode เพื่อส่งออกมาเป็น studentid
    
- **Parameters:**
    
    SQL
    
    ```
    VSTUDENTCODE IN NUMBER
    ```
    
- **Example:** `6831006005`
    

### 📌 GET_COURSE_CAN_W

- **คำอธิบาย:** นับจำนวนรายวิชาที่สามารถถอนได้ในเทอมนั้นๆ
    
- **Parameters:**
    
    SQL
    
    ```
    VSTUDENTCODE IN NUMBER,
    VCOUNT OUT NUMBER
    ```
    
- **Example:** `6`
    

### 📌 GET_STUDENT_INFO

- **คำอธิบาย:** แสดงข้อมูลส่วนตัวของนักศึกษา

    SQL
    
    ```
    VSTUDENTCODE IN NUMBER,
    VACD IN NUMBER,
    VSEM IN NUMBER, 
    VDATA OUT ref_cursor
    ```
    
- **Example:**
    
    Plaintext

    STUDENTCODE 6531006002
    CURRENT_ACADYEAR 2568
    CURRENT_SEMESTER 1
    ACDTEXT First Semester Academic Year 2025
    STUDENTNAME MISS KANYARAT SIANTONG
    FACULTYNAME School of Liberal Arts
    DEPARTMENTNAME English
    PHONENO -
    EMAIL 6531006002@lamduan.mfu.ac.th

- **Parameters:**
    
    SQL
    
    ```
    VSTUDENTCODE IN NUMBER,

    ```
    
- **Example:**

### 📌 ADD_REQUEST

- **คำอธิบาย:** เพิ่มรายวิชาที่ถอนลง `AVSREG.WITHDRAW_REQUEST` มีการตรวจสอบก่อนว่าไม่สามารถถอนรายวิชาออกทั้งหมดได้ จะต้องมีรายวิชาอย่างน้อย 1 วิชา โดยไม่นับรายวิชา MFU333
    
- **Parameters:**
    
    SQL
    
    ```
    VSTUDENTCODE IN NUMBER,
    VACD IN NUMBER,
    VSEM IN NUMBER,
    VCLASS IN NUMBER,
    VCOURSEID IN NUMBER,
    VREASON IN VARCHAR2,
    VMOBILE IN VARCHAR2,
    VEMAIL IN VARCHAR2,
    VDATA OUT ref_cursor,
    VSTATUS OUT NUMBER,
    VMSG OUT VARCHAR2
    ```
    
- **Example:** `2`
