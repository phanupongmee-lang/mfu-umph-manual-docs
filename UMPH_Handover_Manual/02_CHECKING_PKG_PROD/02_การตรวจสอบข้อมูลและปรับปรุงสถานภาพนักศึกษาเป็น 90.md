**ภาพรวม:** ส่วนนี้คือชุดคำสั่งสำหรับการตรวจสอบและปรับปรุงสถานภาพนักศึกษา (Data Manual Adjust) โดยเฉพาะกลุ่มที่ต้องดำเนินการเปลี่ยนสถานะเป็น **90 (ลาออก/พ้นสภาพ)** ซึ่งต้องมีการตรวจสอบความเชื่อมโยงของข้อมูลในหลายส่วนเพื่อป้องกันความผิดพลาด

## 1. คู่มือ: การตรวจสอบข้อมูลและปรับปรุงสถานภาพนักศึกษาเป็น 90 (Manual Status Update)

### 📌 1.1 Overview ของงาน

ในบางกรณีที่ระบบงานปกติไม่สามารถดำเนินการได้ หรือจำเป็นต้องมีการจัดการข้อมูลโดยตรงผ่านฐานข้อมูล เจ้าหน้าที่จะใช้ชุดคำสั่งนี้เพื่อตรวจสอบข้อมูลพื้นฐาน ผลการเรียน และภาระหนี้สินของนักศึกษาก่อนที่จะทำการ **Update สถานภาพเป็น 90**

- **วัตถุประสงค์:** เพื่อตรวจสอบความถูกต้องของข้อมูล (Audit) ในตารางสรุปการลงทะเบียน, ตารางค่าธรรมเนียม และใบแสดงผลการเรียน (Transcript) ก่อนจะดำเนินการเปลี่ยนแปลงสถานะนักศึกษาเข้าสู่สถานะพ้นสภาพ/ลาออก
    

### ✅ 1.2 Checklist ก่อนเริ่มงาน (Pre-requisites)

- **โปรแกรม:** **PL/SQL Developer** เท่านั้น (เนื่องจากมีการเรียกใช้ `rowid` เพื่อการแก้ไข)
    
- **Access Rights:** สิทธิ์ระดับ **Administrator** ที่สามารถใช้คำสั่ง `Update` ใน Schema `AVSREG` ได้
    
- **ความเสี่ยง:** สูง (เนื่องจากเป็นการแก้ไขข้อมูลโดยตรงผ่าน SQL) **ควรทำการ Backup ข้อมูลนักศึกษาที่จะแก้ไขไว้ก่อนทุกครั้ง**
    

### ⚙️ 1.3 ขั้นตอนการทำงาน (Step-by-Step)

#### 🔍 ขั้นตอนที่ 1: ตรวจสอบข้อมูลก่อนแก้ไข (Audit Phase)

ให้ระบุรหัสนักศึกษาลงในวงเล็บ `(studentid)` เพื่อเช็คข้อมูลใน 3 ตารางสำคัญ:

1. **Enroll Summary:** ดูวิชาที่ลงทะเบียนค้างไว้
    
2. **Enroll Fee:** ดูว่ามีหนี้สินค้างชำระหรือไม่
    
3. **Transcript:** ตรวจสอบประวัติการเรียนที่ผ่านมา
    

SQL

```
/* ดึงข้อมูลพร้อม rowid เพื่อตรวจสอบและเตรียมจัดการ */
select t.*, t.rowid from avsreg.enrollsummary t where t.studentid in (/* ใส่รหัสนักศึกษา */); 
select t.*, t.rowid from avsreg.enrollfee t where t.studentid in (/* ใส่รหัสนักศึกษา */); 
select t.*, t.rowid from avsreg.transcript t where t.studentid in (/* ใส่รหัสนักศึกษา */); 
```

#### ✍️ ขั้นตอนที่ 2: การปรับปรุงสถานภาพ (Update Phase)

เมื่อตรวจสอบแล้วว่าข้อมูลถูกต้องและพร้อมดำเนินการ ให้ใช้คำสั่งปรับสถานะเป็น **90** โดยต้องระบุ **ปีการศึกษา และ ภาคเรียน** ที่นักศึกษาเริ่มพ้นสภาพให้ชัดเจน

SQL

```
/* 🚨 ระวัง: ตรวจสอบ WHERE Clause ให้แม่นยำก่อนกด Execute */
update avsreg.studentstatus t 
set t.studentstatus = 90
where t.acadyear = 2564
and t.semester = 1
and t.studentid in (/* ใส่รหัสนักศึกษา */); 
```
