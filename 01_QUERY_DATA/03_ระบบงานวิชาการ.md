## 1. คู่มือ: การตรวจสอบความถูกต้องของหน่วยกิตในโครงสร้างหลักสูตร (Curriculum Integrity Check)

### 📌 1.1 Overview ของงาน

ชุดคำสั่งนี้ใช้สำหรับตรวจสอบ **"ความขัดแย้งของข้อมูลหลักสูตร"** ในกรณีที่โครงสร้างหลักสูตร (Program Structure) ระบุว่าในหมวดวิชานั้นๆ ต้องมีหน่วยกิตรวม (`credittotal > 0`) แต่เมื่อเจาะลึกลงไปในรายวิชาที่ผูกอยู่กับหมวดนั้น (Course in Program) กลับพบรายวิชาที่มีค่าหน่วยกิตเป็น 0

- **วัตถุประสงค์:** เพื่อตรวจหาข้อผิดพลาดในการคีย์ข้อมูลหลักสูตร (Data Entry Error) ก่อนที่ระบบจะนำแผนการศึกษานี้ไปใช้คำนวณการสำเร็จการศึกษาของนักศึกษาจริง หากปล่อยไว้จะทำให้เกณฑ์การนับหน่วยกิตผิดพลาด
    

### ✅ 1.2 Checklist ก่อนเริ่มงาน (Pre-requisites)

- **โปรแกรม:** **PL/SQL Developer**
    
- **Access Rights:** สิทธิ์เข้าถึงตาราง `avsreg.programstructure`, `avsreg.courseinprogram` และ `avsreg.course`
    
- **ข้อมูลที่ต้องเตรียม:** หากต้องการตรวจสอบเฉพาะหลักสูตร สามารถนำรหัสหลักสูตรมาใส่ในเงื่อนไข `and p.programid = 'xxxxxxxxx'` ได้
    

### ⚙️ 1.3 ขั้นตอนการทำงาน (Step-by-Step)

1. เปิดโปรแกรม PL/SQL Developer และ Login
    
2. คัดลอก Code SQL ใน Script 1.3.1 ไปวาง
    
3. **หลักการทำงานของ Script:** * ระบบจะดูที่ตารางโครงสร้างหลักสูตรเป็นตัวตั้ง (`conditiontype = 'C'`)
    
    - จากนั้นจะใช้ `SUBSTR` เพื่อเชื่อมโยงหมวดหลักเข้ากับหมวดย่อย
        
    - ค้นหาหลักสูตรที่มีรายวิชา (`cu.credittotal = 0`) แฝงอยู่ในหมวดที่ควรจะมีหน่วยกิต
        
4. กด Run (Execute)
    
    - **หากพบรายชื่อ Program ID:** ให้เจ้าหน้าที่งานหลักสูตรเข้าไปตรวจสอบที่เล่มหลักสูตร (MCO 2) และแก้ไขหน่วยกิตในระบบให้ถูกต้อง
        

#### 🟢 Script 1.3.1: ตรวจสอบหลักสูตรที่มีรายวิชาหน่วยกิตเป็น 0 ในหมวดที่มีหน่วยกิตรวม

SQL

```
select p.programid 
from avsreg.programstructure p 
where p.conditiontype = 'C' 
and p.credittotal > 0 
and p.programid in 
(
   /* ค้นหาไส้ในของหลักสูตรที่หน่วยกิตรายวิชาผิดปกติ */
   select ps.programid 
   from avsreg.programstructure ps 
   inner join avsreg.courseinprogram ci on ci.programid = ps.programid 
   inner join avsreg.course cu on cu.courseid = ci.courseid
   where SUBSTR(ps.conditionid,1,1) = p.conditionid 
   and ps.conditiontype = 'C' 
   and ps.programid = p.programid 
   and ps.conditionid <> p.conditionid
   and cu.credittotal = 0
)
/* สามารถระบุรหัสหลักสูตรเจาะจงได้ที่นี่ เช่น and p.programid = 685210501 */
group by p.programid;
```
